Return an sd_vector with X percent of 1-bit and an vector<int> of all existing k-mers.

It's not always possible to generate the exact X percent. For example, for a size of 11 we can't produce an sd_vector with exactly 10% of 1. This is why the parameter producedX will store the produced percentage. In this example, it will be of 9.1 percent.

Parameters
len- size of the sd_vector
X- the percentage
producedX- the obtained percentage
Returns
the sd_vector and an int_vector
